PRIVACY POLICY DECLARATION
================================================================================
Advertising ID: NO - App does NOT use advertising ID
Data Collection: NO - App does NOT collect user data
Location: NO - App does NOT access location
Storage: YES - App stores settings locally on device only

Foreground Service: YES
Type: Special Use
Justification: This app provides an accessibility overlay button that helps
users with motor impairments navigate their device. The foreground service is
required to maintain the floating button on screen while the user interacts
with other apps. This is essential for the app's core accessibility functionality.
